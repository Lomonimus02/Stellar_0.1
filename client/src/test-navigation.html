Тест SPA навигации

Инструкции:
  1. Откройте приложение в браузере
  2. Перейдите на несколько разных страниц
  3. Используйте кнопки "Назад" и "Вперед" в браузере
  4. Проверьте, что страница не перезагружается полностью
  5. Убедитесь, что данные остаются кэшированными

Что должно работать:

  • ✅ Кнопки браузера "Назад/Вперед" не перезагружают страницу
  • ✅ Данные остаются в кэше React Query
  • ✅ Навигация происходит мгновенно
  • ✅ URL корректно обновляется
  • ✅ Состояние компонентов сохраняется

Тестовые сценарии:

  1. Базовая навигация: Главная → Пользователи → Классы → Назад → Вперед
  2. Глубокая навигация: Классы → Детали класса → Журнал оценок → Назад
  3. Смена роли: Переключение роли должно использовать SPA навигацию
  4. Обновление данных: Кнопка "Обновить" должна обновлять данные без перезагрузки

Проверка в DevTools:

  1. Откройте DevTools (F12)
  2. Перейдите на вкладку Network
  3. Навигируйте по приложению
  4. При использовании кнопок браузера не должно быть полной перезагрузки HTML
  5. Должны быть только API запросы для новых данных
Успех! Если все тесты прошли, ваше SPA приложение корректно обрабатывает навигацию без перезагрузки страниц.
